# 🧭 Geometry-Based Algorithms !
### 🚀 Master Computational Geometry with Real-World Implementations  

Welcome to **Geometry-Based Algorithms**, a comprehensive collection of **fundamental and advanced geometry algorithms** implemented in both **Java** and **Python**.  
This repository is crafted to help students, developers, and competitive programmers **understand geometry through clean, efficient, and practical code**.

## 🧩 Topics Covered

Each algorithm is implemented in **both Java** and **Python** for cross-language clarity.  
Every topic has been explained visually on my YouTube playlist.

| # | Algorithm | Description | Java File | Python File |
|---|------------|-------------|------------|--------------|
| 100 | **Convex Hull (Graham Scan / Jarvis March)** | Finds the smallest convex polygon enclosing all points. | [`100.ConvexHullAlgo.java`](./100.ConvexHullAlgo.java) | [`100.ConvexHullAlgo.py`](./100.ConvexHullAlgo.py) |
| 101 | **Closest Pair of Points (Divide & Conquer)** | Determines the pair of points with minimum Euclidean distance. | [`101.ClosestPairOfPointsAlgo.java`](./101.ClosestPairOfPointsAlgo.java) | [`101.ClosestPairOfPointsAlgo.py`](./101.ClosestPairOfPointsAlgo.py) |
| 102 | **Rotating Calipers Algorithm** | Used for computing diameters, minimum bounding rectangles, and farthest pairs. | [`102.RotatingCalipersAlgo.java`](./102.RotatingCalipersAlgo.java) | [`102.RotatingCalipersAlgo.py`](./102.RotatingCalipersAlgo.py) |
| 103 | **Line Sweep Algorithm for Segment Intersection** | Detects intersections among multiple line segments efficiently. | [`103.LineSweepAlgorithmforSegmentIntersection.java`](./103.LineSweepAlgorithmforSegmentIntersection.java) | [`103.LineSweepAlgorithmforSegmentIntersection.py`](./103.LineSweepAlgorithmforSegmentIntersection.py) |
| 104 | **Circle-Line and Circle-Circle Intersection** | Computes intersection points between geometric entities. | [`104.Circle-LineAndCircle-CircleIntersectionAlgo.java`](./104.Circle-LineAndCircle-CircleIntersectionAlgo.java) | [`104.Circle-LineAndCircle-CircleIntersectionAlgo.py`](./104.Circle-LineAndCircle-CircleIntersectionAlgo.py) |
| 105 | **Voronoi Diagrams and Delaunay Triangulations** | Constructs Voronoi cells and dual Delaunay triangulations. | [`105.VoronoiDiagramsAndDelaunayTriangulations.java`](./105.VoronoiDiagramsAndDelaunayTriangulations.java) | [`105.VoronoiDiagramsAndDelaunayTriangulations.py`](./105.VoronoiDiagramsAndDelaunayTriangulations.py) |
